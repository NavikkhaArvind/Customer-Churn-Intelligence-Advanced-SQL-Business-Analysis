# Customer Churn Intelligence using SQL

## Project Overview

This project analyzes customer churn data using SQL to identify factors influencing customer retention and revenue. The analysis focuses on customer demographics, churn behavior, revenue trends, and customer lifetime value to generate business insights that support data-driven decision-making.

---

## Objectives

- Analyze customer demographics.
- Measure customer churn rate.
- Identify customer segments with high churn.
- Analyze revenue generated by different customer groups.
- Estimate Customer Lifetime Value (CLV).
- Provide business recommendations to improve customer retention.

# Key Business Insights

- Customers with month-to-month contracts are more likely to churn.
- Customers with lower satisfaction scores show higher churn.
- Long-term customers generate higher customer lifetime value.
- High-value customers contribute significantly to total revenue.
- Customer complaints and support tickets can indicate churn risk.

---

# Business Recommendations

- Encourage customers to switch to long-term contracts.
- Improve customer support for dissatisfied customers.
- Reward loyal customers with retention offers.
- Monitor customers with frequent complaints.
- Design personalized offers for high-risk customers.
